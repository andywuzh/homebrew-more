# homebrew-more

andywuzh's Homebrew tap for deprecated or customized formulae

## install

```bash
brew tap andywuzh/homebrew-more
```

## usage

```bash
brew install <formula>
```

## formulae

  - [x] php@7.1<sup>1</sup>

<sup>1</sup>remove httpd and some other depends, see more via `brew info php@7.1`
